# Orthopedic Surgical Instruments Setup VR Training Simulation

This repository contains the Unity source code for the Bachelor Thesis: **"Virtual Reality Training for Surgical Instrument Setup: Enhancing Medical Education and Operating Room Efficiency"**.

**Author:** Jelena Stefane
**Date:** December 2025
**Unity Version:** 2021.3.44f1

## Description
This project is a Virtual Reality simulation designed to train scrub nurses in surgical instruments layout for orthopedic surgery. It utilizes the XR Interaction Toolkit.

## How to Run the Project
1. Install **Unity Hub** and Unity Version **2021.3.44f1**.
2. Clone or Download this repository.
3. Open Unity Hub, click **Add**, and select the project folder.
4. Open the project.
5. In the Project window, go to `Assets/Scenes` and open **MAIN**.
6. Press Play (requires a VR headset connected).
